toc::[]

== Overview

=== What is Varnish?

Varnish is an upstream tool managed by this plugin. Refer to the official
project documentation for details on usage and capabilities.

=== What is asdf?

{url-asdf}[asdf] is a universal version manager that allows you to manage
multiple runtime versions with a single CLI tool.

=== Why asdf-varnish?

[cols="1,3"]
|===
|Feature |Benefit

|**Version Switching**
|Switch between Varnish versions per project

|**Project Isolation**
|Pin Varnish per repository via `.tool-versions`

|**Reproducible Builds**
|Align CI/CD and developer environments to the same Varnish version

|**Cross-Platform**
|Works on Linux and macOS, with Windows via WSL
|===

=== Dependencies

[source,bash]
----
# Debian/Ubuntu
sudo apt-get update
sudo apt-get install -y curl git

# Fedora/RHEL
sudo dnf install -y curl git

# macOS (via Homebrew)
brew install curl git
----

=== asdf Installation

[source,bash]
----
# Clone asdf
git clone https://github.com/asdf-vm/asdf.git ~/.asdf --branch v0.14.0

# Add to your shell (bash)
echo '. "$HOME/.asdf/asdf.sh"' >> ~/.bashrc
echo '. "$HOME/.asdf/completions/asdf.bash"' >> ~/.bashrc

# Reload your shell
exec $SHELL
----

== Installation

=== Adding the Plugin

[source,bash]
----
asdf plugin add varnish https://github.com/hyperpolymath/asdf-varnish-plugin.git
asdf plugin list
----

=== Installing Varnish Versions

[source,bash]
----
asdf list all varnish
asdf install varnish latest
asdf install varnish 0.1.0
----

=== Setting the Version

[source,bash]
----
asdf global varnish 0.1.0
asdf local varnish 0.1.0
asdf current varnish
varnish --version
----

== Usage

=== Project Configuration

[source]
----
varnish 0.1.0
----

== Configuration

This plugin uses standard asdf behavior. No additional configuration is
required.

== Troubleshooting

* Ensure `varnish` is on PATH after installation.
* Verify the installed version with `varnish --version`.
* Reinstall if a cached release is corrupted.
